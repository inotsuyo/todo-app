# TODOアプリ

HTML/CSS/JavaScriptで作成したシンプルで機能的なTODOアプリケーションです。ローカルストレージを使用してデータを永続化し、優先度、期限、カテゴリ管理などの高度な機能を備えています。

## 機能

### 基本機能
- **タスクの追加**: 新しいタスクを簡単に追加
- **タスクの表示**: 全タスクを見やすく一覧表示
- **完了/未完了の切り替え**: チェックボックスで簡単に状態を変更
- **タスクの削除**: 不要なタスクを削除

### 高度な機能
- **優先度設定**: 3段階の優先度（高・中・低）を設定可能
  - 高優先度: 赤色の境界線
  - 中優先度: オレンジ色の境界線
  - 低優先度: 青色の境界線
- **期限設定**: タスクに期限日を設定
- **カテゴリ/タグ**: タスクをカテゴリで分類・整理
- **フィルタリング機能**:
  - 全て/未完了/完了済みでフィルター
  - カテゴリ別でフィルター
- **データ永続化**: ローカルストレージで自動保存（ブラウザを閉じてもデータが保持）

## 技術スタック

- **HTML5**: セマンティックなマークアップ
- **CSS3**: モダンなデザインとレスポンシブレイアウト
  - グラデーション背景
  - Flexbox/Grid レイアウト
  - スムーズなアニメーション効果
- **JavaScript (ES6+)**: クラスベースの構造
  - LocalStorage API
  - DOM操作
  - イベント処理

## 使い方

### インストール

1. リポジトリをクローン:
```bash
git clone https://github.com/inotsuyo/todo-app.git
cd todo-app
```

2. `index.html` をブラウザで開く:
```bash
open index.html
```

または、ファイルをダブルクリックしてブラウザで開きます。

### 使用方法

1. **タスクの追加**:
   - タスク名を入力
   - 優先度を選択（デフォルト: 中）
   - 必要に応じて期限日とカテゴリを入力
   - 「追加」ボタンをクリック

2. **タスクの完了**:
   - タスク左側のチェックボックスをクリック
   - 完了したタスクは取り消し線が引かれ、半透明になります

3. **タスクの削除**:
   - 各タスクの「削除」ボタンをクリック

4. **フィルタリング**:
   - 「全て」「未完了」「完了済み」ボタンで状態別にフィルター
   - カテゴリドロップダウンでカテゴリ別にフィルター
